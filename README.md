# SMA-Betelgeuse
Group git for the Betelgeuse oort cloud project for SMA

# PROJECT DESCRIPTION:  
We will simulate Betelgeuse and its hypothetical Oort Cloud within the galactic potential as Betelgeuse goes supernova and the Oort Cloud escapes. We want to find out how many (if any) of these objects make it to our solar system.
